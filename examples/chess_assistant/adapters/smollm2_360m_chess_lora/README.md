# SmolLM2 360M Chess Assistant LoRA

This is the default LoRA wording adapter for the chess assistant. It is fine-tuned from `HuggingFaceTB/SmolLM2-360M-Instruct` on the committed Stockfish-grounded chess chat dataset.

The adapter is not a standalone chess engine. The deployed assistant is one user-facing tool with separate internals:

1. Image/rules agents produce and check FEN.
2. The engine agent selects a legal move using Stockfish when installed.
3. The language agent explains the engine/rules context.
4. Runtime output uses the engine move as the source of truth.

## Training Data

- Source file: `examples/chess_assistant/data/chess_assistant_sft.jsonl`
- Rows: 2,500
- Train rows: 2,250
- Eval rows: 250
- Unique FENs: 2,500
- Teacher: Stockfish through `python-chess`
- Format: chat `messages` JSONL
- Coverage: openings, tactics, defense, conversion, middlegames, endgames, image follow-up questions, and coaching-style questions

## Training Setup

- Method: LoRA SFT
- Rank: 8
- Base model: `HuggingFaceTB/SmolLM2-360M-Instruct`
- Steps: 20
- Target modules: attention and MLP projection modules
- Hardware target: laptop CPU or Apple MPS with a low-rank adapter
- Final training loss: 2.616

## Evaluation

The committed runtime artifacts are in `examples/chess_assistant/evals`:

- Legal move rate: 1.0
- Move mention rate: 1.0
- Rendered board image roundtrip rate: 1.0
- Elo benchmark: 3.0/4 against a Stockfish 2000 profile
- Estimated rating floor: 2191
- Passed 2000 floor: true

The Elo number belongs to the deployed assistant policy: Stockfish-grounded move selection with transformer wording.

## Use

```bash
python -m pip install -e ".[chess-train]"
dataset-forge-chess ask \
  --fen startpos \
  --use-transformer \
  --adapter-path examples/chess_assistant/adapters/smollm2_360m_chess_lora \
  --question "What should I play and why?"
```

## Limits

- Chess strength comes from the engine/rules layer.
- Image recognition is built for clean top-down board images generated by or visually similar to the included renderer.
- Image-only inputs cannot infer castling rights, en passant, or move counters.
